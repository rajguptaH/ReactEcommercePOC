# ReactEcommercePOC
# This project is based on react js and strapi 
to run the client app run the below command
```cmd
cd client
npm i
npm start

```
start the server project using
```cmd
cd server
npm i
npm start
```
to change the port of the server project run below command in terminal
```cmd
set PORT={yourport}
```
